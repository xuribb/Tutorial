### 一、数据类型

JavaScript中有8种数据类型，分别如下：

- number

- bigint

- string

- boolean

- undefined

- null

- object（引用类型）

- symbol

我们可以使用`typeof` 来检测变量的类型，示例如下：

```javascript
typeof "abc"   // "string"
typeof(100)     // "number"
```



关于`bigint`类型的说明：`bigint`用于表示任意大小的整型数，可以在数字后面附加`n`来声明一个`bigint`变量。示例如下：

```javascript
let a = 100n;    // 声明一个bigint
let b = 100;     // 声明一个number
a == b           // true
a === b          // false
```

#### 1. ?? 和 || 的比较

`??`是空值合并运算符。`a ?? b `表示如下：

当a不为`null/undefined`的时候返回a,否则返回b

它们之间重要的区别是：

- `||` 返回第一个 **真** 值。
- `??` 返回第一个 **已定义的** 值。
- `||` 无法区分 `false`、`0`、 `""` 和 `null/undefined`。它们都是假值。
